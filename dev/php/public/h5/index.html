什么是 Bootstrap?

Bootstrap是Twitter推出的开源前端开发工具包 ver:2.2.2

“Bootstrap 是我和 Jacob Thornton 编写的一个前端工具箱，目的是为了帮助设计师和 Web 前端开发人员快速有效地创建一个结构简单、性能优良、页面精致的 Web 应用。它使用了最新的浏览器技术，可以提供精致的网页排版方式以及表单、按钮、表格、网格栅格化、导航等诸多元素。”

Mark Otto

Mark Otto
@mto

Jacob Thornton
@fat

优势与劣势

精美的UI
标准化、模块化的设计思路
跨设备、跨浏览器
响应式布局
基于JQUERY的多种插件
基于HTML5及LESS构建
不支持IE6

LESS是什么？

LESS 将 CSS 赋予了动态语言的特性，如 变量， 继承， 运算， 函数. LESS 既可以在 客户端 上运行 (支持IE 6+, Webkit, Firefox)，也可以借助Node.js或者Rhino在服务端运行。

http://www.lesscss.net/

  • 什么是栅格?
  • 为什么是940px
  • 来看看实例>>

  • 目前主流的显示分辨率依然是1024

  • 在IE浏览器下，1024 -（左边框宽度）2px -（右边滚动条的宽度）18px = 1004px

    好无厘头的数字

  • 960能够被12、15、16、24整除，得数分别为80、64、60、40，这些数字听着就和谐而且靠谱。因此960更适合使用栅格设计系统进行网页设计。

  • 960px=n列+(n-1)列间隙+20px侧边间隙

栅格设计系统是一种平面设计的方法与风格。运用固定的格子设计版面布局，其风格工整简洁，在二战后大受欢迎，已成为今日出版物设计的主流风格之一。
